# AI는 +가 아니라 ×다 — AI for English Teachers

A static, print-ready **ebook** on using AI in English teaching. The argument: AI is not an *addition* to your work but a *multiplier* of it — and the multiplier grows through four stages, **Prompt → Context → Harness → Loop**.

**Live:** https://chano-kr.github.io/ai-for-eng-teachers/

## Read it / export to PDF

Open the live page and print to PDF (Ctrl/Cmd + P → paper size **A4** → *Save as PDF*). Print styles paginate by chapter with running heads and page numbers.

## Contents

1. 더하기가 아니라 곱하기 — input of 0 yields 0
2. 네 단계의 지도 — Prompt → Context → Harness → Loop
3. Prompt — its value and its limits
4. Context — the variable that splits the result
5. Harness — design the environment
6. Loop — get better every time
7. The teacher's real workflow — five tasks
8. What to do right now
9. One step further — CLI · Harness · Loop · E2E

## Stack

- Single, self-contained `index.html` (no build step, no framework)
- Type: **Noto Serif KR** (body) + **Pretendard** (labels) via CDN
- `assets/` — two workflow screenshots
- Deployed on GitHub Pages from `main` / root
